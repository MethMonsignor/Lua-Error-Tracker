# Security Policy

## Supported Versions

This module is intended for use with actively maintained Garry's Mod servers. Compatibility with outdated or modified environments is not guaranteed.

## Reporting Vulnerabilities

If you discover a security vulnerability in this module, please open an issue on GitHub with the label `security`. Do not disclose sensitive information publicly.

## Scope

This module does not transmit data externally. All logs are stored locally in `data/error_tracker_log.txt`. Ensure proper file permissions to prevent unauthorized access.
